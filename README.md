# 🧙‍♂️ Strands of Fate 2.0 - Edición Altar Mágico

Plataforma interactiva para la gestión de personajes de rol.

## 🌟 Novedades
- **Altar de Tiradas:** Selector de bonos y lanzador centralizado con animaciones.
- **Historial de Crónicas:** Registro automático de las últimas 5 tiradas con marca de tiempo.
- **Inventario Mágico:** Sistema para añadir objetos con modificadores y descripción.
- **Auto-guardado:** Persistencia total en el navegador mediante LocalStorage.

## 🚀 Uso
Simplemente abre el `index.html` en tu navegador o despliégalo en GitHub Pages.
