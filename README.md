# 📱 Marauder Mobile App - Strands of Fate

Tu grimorio de bolsillo. Esta versión transforma el pergamino en una Progressive Web App (PWA) visual.

## 🌟 Características Mobile
- **Interfaz por Pestañas:** Navega fácilmente entre Perfil, Magia, Dados y Ajustes.
- **Mobile-First:** Botones y campos diseñados para ser pulsados con el pulgar.
- **Estética Inmersiva:** Fondo del Mapa del Merodeador con efectos de desenfoque.
- **Instalable:** Puedes añadirla a la pantalla de inicio de tu móvil desde el navegador.
