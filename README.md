# 🧙‍♂️ Marauder Mobile App - Strands of Fate v6.5

La versión más completa del grimorio digital para **Strands of Fate**. Optimizada para móviles y con gestión total de magia.

## 🌟 Novedades v6.5
- **🎒 Reliquias y Objetos:** Nueva sección dinámica en la pestaña de Magia para gestionar objetos con modificadores numéricos.
- **📖 Libro de Hechizos:** Espacio dedicado para anotar hechizos aprendidos durante la campaña con descripciones detalladas.
- **📱 Interfaz Unificada:** Navegación por pestañas fluida con diseño inmersivo.
- **📜 Persistencia Total:** Todos los datos (incluyendo hechizos y objetos) se exportan e importan correctamente en JSON.

## 🚀 Instalación
Sube `index.html`, `style.css`, `app.js` y `oath-of-the-marauders-map-rty5hmjpco2j92s6.jpg` a tu repositorio de GitHub y activa **GitHub Pages**.
