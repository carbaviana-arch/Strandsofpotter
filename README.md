# 🧙‍♂️ Marauder Mobile App - Strands of Fate v6.1

Esta es la evolución definitiva del grimorio digital para el sistema de rol **Strands of Fate**. Diseñada como una aplicación web (PWA) bajo la estética del "Mapa del Merodeador", ofrece una experiencia inmersiva, profesional y totalmente optimizada para dispositivos móviles.

## 🌟 Características de la Versión 6.1
- **📱 Interfaz de App Nativa:** Navegación por pestañas mediante un Dock estilizado con efecto *Glassmorphism*.
- **🛡️ Sistema de Vínculos Dinámicos:** Gestión total de aspectos y vínculos con modificadores numéricos editables en tiempo real.
- **✨ Artes Arcanas Avanzadas:** Control de habilidades con cálculo automático de "Esencia" basado en el valor de Recuperación.
- **🎲 Altar de Adivinación:** Sistema de tiradas 4dF (Fate/Fudge) con selector de habilidades e historial integrado.
- **🩹 Trackers de Estrés Profesional:** Seguimiento de daño físico y mental ampliado a 5 casillas con campos de consecuencias independientes.
- **📜 Persistencia de Datos:** Capacidad para sellar (exportar) y leer (importar) el grimorio en formato JSON para no perder nunca el progreso.

## 🚀 Instalación Rápida
Para desplegar tu propia versión en **GitHub Pages**:

1. **Clona o descarga** este repositorio.
2. Asegúrate de que los siguientes archivos estén en la carpeta raíz:
   - `index.html`
   - `style.css`
   - `app.js`
   - `oath-of-the-marauders-map-rty5hmjpco2j92s6.jpg` (Imagen de fondo).
3. En GitHub, ve a **Settings > Pages**.
4. En la sección "Build and deployment", elige la rama `main` y pulsa **Save**.
5. Tu grimorio estará listo en: `https://tu-usuario.github.io/Strandsofpotter/`

## 🕹️ Guía de Uso
- **Perfil:** Haz clic en el botón `+` para añadir nuevos vínculos o aspectos. Usa los campos numéricos para bonos temporales.
- **Magia:** El contador de "Esencia Libre" se actualiza automáticamente al asignar niveles a tus Artes Arcanas.
- **Juego:** Selecciona el arte que deseas usar y pulsa **INVOCAR**. El resultado aparecerá instantáneamente en la esfera.
- **Archivo:** Escribe tus crónicas y guarda tu progreso descargando el archivo del pergamino.
