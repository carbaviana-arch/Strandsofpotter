# 🧙‍♂️ Marauder Mobile App - Strands of Fate v6.6

La versión definitiva del grimorio digital para **Strands of Fate**. Optimizada para móviles, con gestión total de magia y código de colores para daños.

## 🌟 Novedades v6.6
- **🛑 Código de Colores en Consecuencias:** Los campos de daño Leve, Moderado y Grave ahora tienen fondos rojizos de intensidad creciente para una identificación visual rápida.
- **🎒 Reliquias y Objetos:** Sección dinámica para gestionar objetos mágicos con modificadores numéricos.
- **📖 Libro de Hechizos:** Espacio para anotar hechizos aprendidos con descripciones detalladas.
- **📱 Interfaz Unificada:** Navegación por pestañas fluida con diseño inmersivo.
- **📜 Persistencia Total:** Guardado y carga en JSON actualizado para incluir las consecuencias coloreadas.

## 🚀 Instalación
Sube `index.html`, `style.css`, `app.js` y `oath-of-the-marauders-map-rty5hmjpco2j92s6.jpg` a tu repositorio de GitHub y activa **GitHub Pages**.
